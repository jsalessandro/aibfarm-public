## 🚀 安装说明

### Android 安装
1. 下载 `.apk` 文件
2. 允许"未知来源"应用安装
3. 点击安装文件完成安装

### iOS 安装
1. 下载 `.ipa` 文件
2. 使用 AltStore、Sideloadly 或其他工具进行侧载安装
3. 信任开发者证书
